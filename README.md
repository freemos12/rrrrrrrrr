# rrrrrrrrr
# Random Task Generator

**Автор:** Семён Воробьев
## Описание

Приложение для генерации случайных задач с фильтрацией по типу (учёба, спорт, работа), сохранением истории и возможностью добавления собственных задач.

## Возможности

- Генерация случайной задачи из предопределённого списка.
- Отображение истории сгенерированных задач.
- Фильтрация задач по типу.
- Сохранение истории в JSON.
- Добавление новых задач с проверкой ввода.
- Использование Git для контроля версий.

## Примеры использования

1. Запустите main.py.
2. Нажмите «Сгенерировать задачу» — появится случайная задача.
3. Используйте фильтр для просмотра задач по типу.
4. Добавьте свою задачу через форму.
5. История сохраняется между запусками.
